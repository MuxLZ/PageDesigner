# 页面配置预览指南

## 概述

本指南介绍如何将导出的页面配置在浏览器中正常显示。

## 使用方法

### 方法一：导出并预览（推荐）

1. 在设计器中完成页面设计
2. 点击工具栏中的 **"导出并预览"** 按钮
3. 系统会自动：
   - 导出配置文件（JSON格式）
   - 跳转到预览页面显示页面效果

### 方法二：手动上传配置

1. 在设计器中点击 **"导出配置"** 按钮，下载JSON配置文件
2. 访问预览页面：`/preview`
3. 将下载的JSON文件拖拽到上传区域，或点击上传
4. 页面会自动加载并显示配置

### 方法三：通过URL参数

如果配置数据已经在URL参数中，可以直接访问：
```
/preview?config=<编码后的JSON配置>
```

### 方法四：从本地存储加载

如果配置已保存到浏览器的本地存储，可以直接点击 **"从本地存储加载"** 按钮。

## 预览页面功能

预览页面提供以下功能：

- **页面渲染**：完整显示您设计的页面效果
- **下载配置**：重新下载当前配置的JSON文件
- **重新加载**：清空当前配置，重新上传
- **在设计器中编辑**：跳转到设计器继续编辑

## 注意事项

1. **配置文件格式**：确保上传的是有效的JSON格式配置文件
2. **模块依赖**：确保所有使用的模块都已正确注册
3. **资源路径**：如果配置中包含图片或其他资源，确保路径正确
4. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Edge等）

## 常见问题

### Q: 预览页面显示空白？
A: 检查配置文件格式是否正确，或者尝试重新导出配置。

### Q: 某些模块没有正常显示？
A: 确保模块配置完整，检查是否有必需的字段缺失。

### Q: 样式显示不正确？
A: 检查全局样式配置，确保主题颜色等配置正确。

### Q: 如何保存预览的配置？
A: 在预览页面点击"下载配置"按钮，可以保存当前配置。
