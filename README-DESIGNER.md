# 页面设计器使用文档

## 概述

这是一个低代码页面设计系统，允许您通过可视化界面快速搭建和定制网站页面。系统支持模块化组件、样式配置、动画效果和实时预览。

## 功能特性

### 1. 模块化架构
- **模块类型**：支持多种模块类型（轮播、宫格、分栏、时间轴、导航栏、页脚等）
- **可扩展性**：易于添加新模块类型
- **模块独立性**：每个模块都有独立的配置和样式

### 2. 可视化配置
- **拖拽排序**：调整模块在页面中的顺序
- **实时预览**：配置更改立即反映在预览中
- **属性面板**：为每个模块提供详细的配置选项

### 3. 样式系统
- **主题配置**：全局颜色主题设置
- **字体配置**：字体族、大小、颜色、行高等
- **间距配置**：内边距、外边距、模块间距
- **布局配置**：网格布局、弹性布局等

### 4. 动画效果系统
- **多种效果**：淡入淡出、滑动、缩放、旋转、弹跳、翻转
- **效果组合**：支持多个效果组合使用
- **可配置参数**：持续时间、延迟、缓动函数等

## 使用指南

### 访问设计器

访问路由 `/designer` 即可进入页面设计器。

### 添加模块

1. 在左侧"模块库"中选择要添加的模块类型
2. 点击模块名称即可添加到页面
3. 模块会自动添加到模块列表末尾

### 配置模块

1. 在模块列表中点击要配置的模块
2. 右侧属性面板会显示该模块的配置选项
3. 配置包括：
   - **基础设置**：模块名称、可见性、排序
   - **样式设置**：背景色、内边距、外边距、字体等
   - **动画效果**：效果类型、持续时间、延迟
   - **模块配置**：模块特定的配置项

### 调整模块顺序

1. 在模块列表中直接拖拽模块项
2. 或通过调整"排序"数值来改变顺序

### 显示/隐藏模块

1. 在模块列表中切换模块的开关
2. 或在属性面板中设置"是否显示"

### 全局样式配置

1. 在左侧"页面设置"中配置全局主题
2. 包括主色、次色、背景色等
3. 更改会应用到整个页面

### 保存和导出

1. **保存配置**：点击工具栏的"保存配置"按钮，配置会保存到本地存储
2. **导出配置**：点击"导出配置"按钮，将配置导出为JSON文件
3. **导入配置**：点击"导入配置"按钮，从JSON文件导入配置

## 模块类型说明

### Header（导航栏）
- Logo配置（图片/文字）
- 导航菜单项
- 顶部联系栏
- 固定/透明选项

### Carousel（轮播）
- 轮播图片/内容
- 自动播放设置
- 指示器和箭头
- 切换效果

### Grid（宫格）
- 网格布局
- 列数配置
- 项目内容

### Column（分栏）
- 分栏布局
- 图片/文字配置
- 布局方式（左图右文/右图左文/交替）

### Timeline（时间轴）
- 时间线项目
- 布局方式（垂直/水平/交替）
- 时间线和标记颜色

### Stats（数据统计）
- 统计数据项
- 数字显示（前缀/后缀）
- 列数配置

### Footer（页脚）
- 页脚区块
- 链接配置
- 社交媒体链接
- 版权信息

## 样式配置

### 颜色主题
```typescript
{
  primary: '#1e3a8a',      // 主色
  secondary: '#3b82f6',   // 次色
  accent: '#f59e0b',      // 强调色
  background: '#ffffff',   // 背景色
  text: '#333333'         // 文字色
}
```

### 字体配置
```typescript
{
  family: "'PingFang SC', 'Microsoft YaHei', sans-serif",
  size: '16px',
  weight: 'normal',
  color: '#333',
  lineHeight: '1.6'
}
```

### 间距配置
```typescript
{
  top: '20px',
  right: '20px',
  bottom: '20px',
  left: '20px',
  // 或使用简写
  horizontal: '20px',
  vertical: '20px'
}
```

## 动画效果

### 单个效果
- `fade` - 淡入淡出
- `slide` - 滑动（可指定方向：up/down/left/right）
- `zoom` - 缩放
- `rotate` - 旋转
- `bounce` - 弹跳
- `flip` - 翻转（可指定方向）

### 组合效果
- `fadeInUp` - 淡入+向上滑动
- `fadeInZoom` - 淡入+缩放
- `slideZoom` - 滑动+缩放

### 效果参数
- `duration` - 持续时间（毫秒）
- `delay` - 延迟时间（毫秒）
- `easing` - 缓动函数（如：ease-in-out）

## 扩展开发

### 添加新模块类型

1. 在 `src/types/module.ts` 中添加新的模块类型枚举
2. 定义模块配置接口
3. 创建模块组件（参考现有模块组件）
4. 在 `PageRenderer.vue` 中注册模块组件
5. 在 `ConfigGenerator.ts` 中添加默认配置生成逻辑

### 添加新动画效果

1. 在 `src/types/module.ts` 中添加新的效果类型
2. 在 `src/core/EffectSystem.ts` 中实现效果创建方法
3. 在 `src/assets/styles/effects.css` 中添加CSS动画

## 技术栈

- Vue 3 + TypeScript
- Element Plus UI组件库
- Pinia 状态管理
- Vue Router 路由
- SortableJS 拖拽排序
- AOS 动画库

## 注意事项

1. 配置更改会实时反映在预览中，但不会自动保存
2. 建议定期保存配置，避免数据丢失
3. 导出配置可以用于备份或在其他项目中复用
4. 模块样式配置支持CSS标准值（如：'20px', '1rem', '10%'等）
