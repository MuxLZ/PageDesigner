# 快速开始指南

## 安装和运行

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **访问设计器**
   打开浏览器访问：`http://localhost:5173/designer`

## 第一个页面

### 步骤1：添加导航栏

1. 在左侧"模块库"中点击"导航栏"
2. 在右侧属性面板中配置：
   - Logo文字：输入您的网站名称
   - 菜单项：添加导航链接
   - 固定导航：开启固定选项

### 步骤2：添加轮播

1. 点击"轮播"模块
2. 配置轮播内容：
   - 添加轮播图片
   - 设置标题和副标题
   - 配置按钮

### 步骤3：添加内容模块

1. 添加"宫格"模块展示产品/服务
2. 添加"数据统计"模块展示数据
3. 添加"分栏"模块展示详细介绍

### 步骤4：添加页脚

1. 添加"页脚"模块
2. 配置页脚链接和版权信息

### 步骤5：保存和导出

1. 点击"保存配置"保存到本地
2. 点击"导出配置"导出JSON文件

## 常用配置

### 全局主题设置

在左侧"页面设置"中：
- 主色：网站主要颜色
- 次色：次要强调色
- 背景色：页面背景色

### 模块样式

选中模块后，在右侧属性面板：
- **基础设置**：模块名称、可见性、排序
- **样式设置**：背景色、内边距、字体
- **动画效果**：效果类型、持续时间、延迟

### 模块配置

根据模块类型不同，会有不同的配置项：
- **轮播**：高度、自动播放、指示器
- **宫格**：列数、项目内容
- **统计**：统计数据、列数
- **时间轴**：时间线项目、布局方式

## 常见问题

### Q: 如何调整模块顺序？
A: 在模块列表中点击模块，然后在属性面板中修改"排序"数值，数值越小越靠前。

### Q: 如何隐藏某个模块？
A: 在模块列表中关闭模块的开关，或在属性面板中取消"是否显示"。

### Q: 配置如何保存？
A: 点击工具栏的"保存配置"按钮，配置会保存到浏览器本地存储。也可以导出为JSON文件。

### Q: 如何导入已有配置？
A: 点击工具栏的"导入配置"按钮，选择之前导出的JSON文件即可。

### Q: 如何添加自定义模块？
A: 参考 `ARCHITECTURE.md` 中的扩展指南，按照步骤添加新模块类型。
